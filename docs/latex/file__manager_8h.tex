\doxysubsection{Detailed Description}
Header file per la gestione dei file binari del sistema. 

\begin{DoxyAuthor}{Author}
Francesco Pice -\/ Francesco Sblendorio 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
23-\/06-\/2025 
\end{DoxyDate}
\begin{DoxyVersion}{Version}
1.\+0
\end{DoxyVersion}
Questo modulo fornisce funzionalità complete per la gestione della persistenza dei dati attraverso file binari, gestendo sia i videogame che le recensioni con operazioni di lettura, scrittura e modifica.
