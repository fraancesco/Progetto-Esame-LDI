Detailed Description

Header file per la ricerca e l'ordinamento dei videogame.

Author
Fractesco Pice - Francesco Sblendorio
Date
23-06-2025
Version
1.0

Questo modulo fornisce funzionalità complete per la ricerca, il filtraggio e l'ordinamento dei videogame, con interfacce differenziate per amministratori e utenti.
